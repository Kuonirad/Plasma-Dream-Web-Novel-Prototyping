# Contributing to Plasma Dream-Web

Thank you for your interest in contributing to the Plasma Dream-Web project.

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/your-feature`)
3. **Commit** your changes (`git commit -am 'Add feature'`)
4. **Push** to the branch (`git push origin feature/your-feature`)
5. **Open** a Pull Request

## Areas of Contribution

- **Hardware prototyping:** Build any of the 7 prototypes and share results
- **Digital twin:** Improve WebGL2 spectrogram, add features, optimize performance
- **Bayesian controller:** Refine control algorithms, add new control modes
- **Safety:** Identify risks, improve safety protocols, add regulatory data
- **Documentation:** Improve clarity, add tutorials, translate
- **CAD files:** Create or improve mechanical designs for prototypes

## Safety First

All contributions involving hardware must include updated safety documentation. No PR will be merged without safety review.

## Code of Conduct

Be respectful, collaborative, and scientifically rigorous. All claims must be evidence-based.
